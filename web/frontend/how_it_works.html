Understanding the Neural Architecture from raw signal to diagnosis.

Raw Data Acquisition

The process begins by capturing bioelectrical activity using a 14-channel EEG headset (Emotiv/OpenBCI). The sensors record micro-voltage fluctuations on the scalp resulting from ionic current within the brain's neurons.

Signal Preprocessing

Raw EEG is noisy. We apply a Band-Pass Filter (1-40Hz) to isolate relevant brainwaves (Alpha, Beta, Theta) and remove artifacts like muscle movement (EMG) or power line interference (60Hz).

Raw ‚ûù Filter(1-40Hz) ‚ûù Clean Signal

Tensor Construction

The clean signal is sliced into 4-second windows and transformed into a 3D Tensor. This multi-dimensional matrix represents Channels √ó Frequency √ó Time, allowing the AI to analyze spatial and spectral features simultaneously.

Input Shape: [14, 33, 17]
4
üíé

Feature Embedding

A Siamese Neural Network processes the tensor. Instead of classifying directly, it maps the input into a high-dimensional vector space called an "Embedding". This vector is the unique mathematical fingerprint of the patient's mental state.

Tensor ‚ûù CNN Encoder ‚ûù Vector
5

Few-Shot Comparison

Using Prototypical Networks, the system calculates the Euclidean Distance between the new patient's embedding and the learned "Prototypes" (centroids) of Relaxed vs. Burnout classes derived from the training set.

d(x, Prototype_A) vs d(x, Prototype_B)
6

XAI & Diagnosis

The final diagnosis is based on the nearest prototype. Simultaneously, Grad-CAM generates a heatmap (Explainable AI), highlighting exactly which brain regions and frequencies (e.g., Beta waves in Frontal Lobe) triggered the decision.

Output: Burnout (98%) + Heatmap
